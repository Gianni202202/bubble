# How to Run Your React Website

This React project contains your 4 design screens converted into code.

### Step 1: Open Terminal
1.  Open your Terminal app.

### Step 2: Run the Project
Copy and paste this command to install dependencies and start the development server:

```bash
cd "/Users/giannilinssen/.gemini/antigravity/scratch/elvatix-homepage-react" && \
npm install && \
npm run dev
```

### Step 3: View in Browser
- The terminal will show a link (usually `http://localhost:5173`).
- Click it (Cmd+Click) to open your new website!

### Features
- **Routing**: Click the navigation links to switch between Homepage, Features, TaskFlow, and Industry pages.
- **Tailwind CSS**: Fully responsive and styled.
- **Vite**: Super fast hot-reloading.
